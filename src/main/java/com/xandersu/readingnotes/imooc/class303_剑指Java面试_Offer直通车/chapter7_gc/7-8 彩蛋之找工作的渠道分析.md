# 7-8 彩蛋之找工作的渠道分析

## 同事朋友的内推

- 简历直达团队，避免被非团队的人员筛掉
- 知己知彼
  - 大致的薪酬范围
  - 内部是否缺人
  - 面试进度
- 避免简历被锁
- 注意：碍于情面，错过后面的好公司



## 招聘网站的投递

#### 最不推荐

- 简历会锁定
- 太多的虚假信息
  - 看官网



## 猎头招聘

#### 人性最真实的一面

- 猎头对特定的公司、职位比较熟悉
- 找干练的猎头：是否做到精准推送，建议足够好
- 找多个猎头，多方面了解市场
- 做好恶战的准备
- 搭线练手的公司，查漏补缺

## 总结

- 利用猎头对公司进行分类：练手公司和心仪公司
- 竭尽全力挖掘目标公司的内部员工
