## DHCP协议详解 

- DHCP(Dynamic Host Configuration Protocol: 动态主机设置协议) 
- DHCP是一个局域网协议
- DHCP是应用UDP协议的应用层协议 



- DHCP服务器监听默认端口:67
- 主机使用UDP协议广播DHCP发现报文 
- DHCP服务器发出DHCP提供报文
- 主机向DHCP服务器发出DHCP请求报文 
- DHCP服务器回应并提供IP地址
